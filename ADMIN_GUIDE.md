## 1. 登录后台

1. 打开浏览器，访问 **https://www.zxpapers.com/admin**

## 3. 图片管理 — 替换全站任何图片

> 这是老板最常用的功能：**不动代码就能换网站任何一张图**

### 怎么用

1. 左侧菜单点 **「图片管理」**
2. 页面会按位置分组显示所有图片槽位（首页、关于、产品页…）
3. 找到要换的图，点 **「上传新图」**
4. 选择本地图片（推荐 JPG/WebP，**单张 < 2MB**，宽度 1200~2400px）
5. 上传成功后，**网站会自动刷新**（约 5 秒生效）

### 想换回原图？

点对应卡片的 **「重置为默认」**，立刻恢复设计师做的原图。

### 进阶玩法

- 顶部过滤器：**「已自定义」** 可以快速查看你换过哪些图
- 点 **「在网站查看」** 按钮，会直接跳到该图所在的前台页面

> 📷 **图片为什么这么快？** 所有图片走 Cloudflare R2 全球 CDN，国内外都飞快。

## 4. 文章管理 — 写博客做 SEO

> 博客是 Google 排名的关键武器，越多原创高质量文章，自然流量越多。

### 写新文章

1. 左侧菜单 → **「文章管理」** → 右上角 **「写新文章」**
2. 填写：
   - **标题**：吸引人，包含关键词（例：`How to Choose the Right Thermal Paper Roll for Your POS Printer`）
   - **摘要**：1~2 句话，会显示在文章列表
   - **封面图**：右侧上传，会被 Google 抓取做缩略图
   - **正文**：支持 Markdown 语法（见下方速查表）
   - **SEO 标题/描述/关键词**：右侧侧栏填写，针对 Google
3. 点 **「保存草稿」** 暂存，或 **「立即发布」** 上线

### Markdown 速查表（傻瓜版）

```markdown
# 一级大标题
## 二级标题
### 三级小标题

正文段落，直接写。

**加粗的字**   *斜体的字*

- 列表项 1
- 列表项 2

1. 步骤 1
2. 步骤 2

[蓝色链接](https://example.com)

![图片描述](图片URL)

> 这是一个引用，会显示成蓝色横条

`代码片段`

---

（三个减号 = 分割线）
```

### 文章 SEO 怎么填？

| 字段 | 建议 |
|------|------|
| SEO 标题 | 60 字符内，包含主关键词，例：`Thermal Paper Roll Buying Guide 2026 - ZhixinPaper` |
| SEO 描述 | 150~160 字符，吸引用户点击 |
| 关键词 | 3~5 个，用英文逗号分隔 |

### 已发布的文章访问地址

`https://www.zxpapers.com/blog/你的slug`
比如 slug 是 `thermal-paper-guide`，访问就是 `/blog/thermal-paper-guide`

## 5. 客户询盘 — 处理订单咨询

### 询盘从哪来？

客户在网站任何 **联系/询价表单** 提交后，会**同时**：

1. ✅ 保存到后台「客户询盘」（永久留底）
2. ✅ 发送邮件到你的邮箱（通过 Web3Forms）
3. ✅ 推送到 **企业微信** / **飞书** / **Server酱（微信）**（配置见第 7 节）

### 在后台怎么处理？

1. 左侧菜单 → **「客户询盘」**
2. 看到一条新询盘（红色 NEW 徽章），点开查看详情
3. 操作按钮：
   - **回复邮件** — 一键打开邮件客户端
   - **标记已处理** — 不再显示在"未处理"
   - **归档** — 收起到归档区
   - **删除** — 永久移除（慎用）

### 过滤器

顶部可按 **全部 / 未处理 / 已处理 / 已归档** 切换。

## 6. SEO 设置 — 让 Google 找到你

左侧菜单 → **「SEO 设置」**，分 4 个区域：

### ① 全站基础信息
- **站点标题**：浏览器标签页显示的名字
- **默认描述**：没单独设置 SEO 的页面会用这个
- **默认 OG 图**：分享到社交媒体的预览图（建议 1200×630）

### ② 搜索引擎验证（关键！）

去这些地方申请验证码，复制到对应输入框：

| 平台 | 申请地址 | 用途 |
|------|---------|------|
| **Google Search Console** | https://search.google.com/search-console | 监控 Google 收录 / 排名 |
| **Bing Webmaster** | https://www.bing.com/webmasters | Bing 搜索 |
| **百度站长** | https://ziyuan.baidu.com/ | 百度收录（针对国内） |

验证方式选 **HTML meta 标签**，把 `content="xxxxx"` 里的 `xxxxx` 复制进来即可。

### ③ 第三方追踪

| 工具 | 在哪填 | 怎么获取 |
|------|--------|---------|
| **Google Analytics** | `G-XXXXXXXXXX` | https://analytics.google.com → 管理 → 数据流 |
| **Google Tag Manager** | `GTM-XXXXXX` | https://tagmanager.google.com |
| **Facebook Pixel** | 15~16 位数字 | https://business.facebook.com → 事件管理工具 |

填好后**点保存**，全站会自动注入跟踪代码。

### ④ 通知渠道状态

显示企业微信/飞书/Server酱/Web3Forms 是否已配置好。**没绿勾的赶紧配（见第 7 节）**。

## 7. 配置消息通知（重要！）

> 客户填表后，你要在 **微信 / 企业微信 / 飞书** 立刻收到提醒，不能漏单。

这些配置在服务器的 **环境变量**里，需要登录服务器/Vercel 控制台修改。

### 方法一：企业微信（推荐 — 团队协作好用）

1. 用企业微信打开任意群 → 右上角 **…** → **群机器人** → **添加机器人**
2. 选 **"自定义机器人"**，给个名字（例：`zxpaper询盘`）
3. 复制生成的 **Webhook URL**（类似 `https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx`）
4. 在服务器的环境变量里加：`WECOM_WEBHOOK_URL=粘贴这里`

### 方法二：飞书

1. 飞书群 → **群设置** → **群机器人** → **添加机器人**
2. 选 **"Custom Bot"（自定义机器人）**
3. 复制 Webhook URL
4. 环境变量：`FEISHU_WEBHOOK_URL=粘贴这里`

### 方法三：Server酱（个人微信推送，免费）

1. 访问 https://sct.ftqq.com/ 用微信扫码登录
2. 复制你的 **SendKey**（类似 `SCT123456Txxxxxxx`）
3. 环境变量：`SERVERCHAN_SENDKEY=粘贴这里`

### 方法四：邮件（Web3Forms — 已默认接入）

代码里默认 access key 是 `5dfa9b0f-2f76-4ce6-b30b-d5b91c5b5e69`，
如果想用自己的：
1. 注册 https://web3forms.com/
2. 创建 Access Key
3. 环境变量：`WEB3FORMS_ACCESS_KEY=粘贴这里`

**💡 这 4 个通道可以同时开，开几个收几份**，不会冲突。

## 8. 安全设置：修改密码

**默认密码 `zxpaper2026` 必须改！**

在服务器/Vercel 环境变量里：
```
ADMIN_PASSWORD=你的新密码至少12位含字母数字
ADMIN_SESSION_SECRET=随便敲一串64位的乱码
```

`ADMIN_SESSION_SECRET` 推荐生成方式（任选一）：
- 在线生成：https://www.random.org/strings/?num=1&len=64&digits=on&upperalpha=on&loweralpha=on&format=html
- 命令行：`openssl rand -hex 32`

改完后**重新部署**生效，旧的登录态会全部失效（你需要重新登录）。

## 9. 部署上线时的注意事项

> 当前项目使用 **本地 JSON 文件** 存储数据（`data/*.json`）。

### 如果部署到 Vercel：
⚠️ Vercel 的文件系统是**只读**的，本地 JSON 方案**不能直接用**！需要任选一种迁移：

| 方案 | 难度 | 成本 | 推荐 |
|------|------|------|------|
| **Vercel KV** | 简单 | 免费层够用 | ⭐⭐⭐⭐⭐ |
| **Vercel Blob** | 简单 | 免费层 1GB | ⭐⭐⭐⭐ |
| **Supabase** | 中等 | 免费 500MB | ⭐⭐⭐⭐ |
| **保留 Sanity** | 已配置 | 免费 100k 文档 | ⭐⭐⭐ |

> 💡 跟开发者说"把 data/*.json 切到 Vercel KV"即可，代码改动很小。

### 如果部署到自有服务器（VPS / 阿里云）：
✅ JSON 文件方案直接能用，**记得定期备份 `data/` 目录**。

推荐每天凌晨自动备份脚本（添加到 crontab）：
```bash
0 3 * * * cd /path/to/zxpaper && tar czf backup-$(date +\%Y\%m\%d).tar.gz data/
```

## 10. 常见问题 FAQ

### Q1：换了图片为什么没立刻显示？
A：浏览器有缓存。**强制刷新**：Windows 按 `Ctrl+F5`，Mac 按 `Cmd+Shift+R`。

### Q2：忘记后台密码怎么办？
A：找开发者修改 `.env.local` 里的 `ADMIN_PASSWORD`，重启服务。

### Q3：能多人同时用后台吗？
A：可以，但都用同一个密码登录。如果以后要多人多权限，告诉开发者升级到多账号系统。

### Q4：客户询盘会丢吗？
A：不会。三层保护：
1. 后台 `data/inquiries.json` 永久保存
2. 邮件副本到你邮箱
3. 即时推送到微信/飞书

### Q5：博客 Markdown 太复杂，能给我富文本编辑器吗？
A：当前为了零依赖用的纯 Markdown。如果你不喜欢，告诉开发者升级到 Tiptap/TinyMCE 富文本（需要额外安装包）。

### Q6：怎么知道我的 SEO 做得好不好？
A：
1. 登录 Google Search Console 看 **「效果」** 报告
2. 用 Ahrefs / SEMrush 第三方工具
3. 直接 Google 搜你的关键词看排名

### Q7：网站打开慢怎么办？
A：检查：
- 图片是不是太大（每张控制 < 500KB）
- 是否启用了 CDN（你已经用了 R2，应该不慢）
- 找开发者跑一下 PageSpeed Insights https://pagespeed.web.dev/
