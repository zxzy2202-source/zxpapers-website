# 10 产品知识包

这里维护稳定的产品事实和选型逻辑，不放一次性市场报告或泛营销文案。

## 产品线结构

- [11-Thermal-Paper-Rolls](11-Thermal-Paper-Rolls/README.md)
- [12-Thermal-Labels](12-Thermal-Labels/README.md)
- [13-Printed-and-Product-Labels](13-Printed-and-Product-Labels/README.md)

每条产品线采用同一组核心文档：

| 文档 | 必须回答 |
|---|---|
| `00-产品线索引` | 当前有效内容、Owner、复查日期 |
| `01-品类与SKU结构` | 产品如何分组，哪些应合并或独立 |
| `02-规格材料与适配` | 尺寸、材料、设备、环境和兼容边界 |
| `03-选型与报价字段` | 报价前必须收集什么，如何避免错配 |
| `04-质量控制与失效模式` | 常见问题、测试方法和验收标准 |
| `05-包装交付与OEM` | 包装、印刷、交期、MOQ 和定制流程 |
| `06-禁止承诺与升级条件` | 不能直接承诺什么，何时必须升级审核 |

## 迁移优先级

- P0：规格适配矩阵、报价字段、选型、质量失效、包装/OEM。
- P1：特种产品、环保化学体系、行业专用材料。
- P2：宽泛市场综述和历史汇总，仅保留链接或归档。

## 边界

- 关键词和页面结构归 `40-网站与增长`。
- 客户画像和区域机会归 `20-市场与客户`。
- 证书、测试、法规和客户案例证据归 `50-证据与合规`。
