# ZXPaper 知识库

本目录是 ZXPaper 独立站的本地知识源骨架。它不是旧飞书知识库的整库复制，而是把旧库拆成可供产品、销售、运营和管理共同使用的模块。

## 建库原则

1. 一份正式结论只保留一个权威正文，其他工作台只链接，不复制。
2. 产品事实、市场判断、销售动作、页面内容和证据记录分层维护。
3. 飞书用于阅读、协作和评论；本地 Markdown 作为可版本化的内容源。
4. 所有对外能力声明必须能回到 `50-证据与合规` 的证据记录。
5. 旧库内容先建立迁移映射，再分批吸收；不直接把 417 个节点原样搬入新库。
