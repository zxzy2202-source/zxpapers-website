# 50 证据与合规

这里是对外声明的审核入口。目标不是堆放证书，而是让每一句关键卖点都能判断“能否使用、适用于什么条件、何时失效”。

## 标准结构

| 模块 | 内容 |
|---|---|
| `51-声明登记` | Claim ID、原文、适用产品、状态、Owner |
| `52-产品与化学合规` | BPA/BPS/phenol、REACH、RoHS、食品接触等 |
| `53-测试与质检` | 方法、条件、样本、结果、日期、设备 |
| `54-证书与供应商文件` | TDS、SDS、COA、证书、有效期 |
| `55-设备与产能` | 设备、幅宽、速度、产能、可验证边界 |
| `56-项目与客户证据` | 可公开案例、匿名项目记录、授权范围 |
| `57-来源与复查队列` | 外部来源、Needs Validation、到期提醒 |

## 声明状态

| 状态 | 使用规则 |
|---|---|
| Verified | 在记录的产品、测试和时间范围内可使用 |
| Conditional | 必须带条件、方法或产品范围 |
| Needs Validation | 不能作为对外确定性承诺 |
| Rejected | 已证实不准确或不适用于本公司 |
| Expired | 已过有效期，复查前暂停使用 |

## 最低证据字段

`Claim ID / 声明 / 产品或SKU / 测试或来源 / 条件 / 结果 / 状态 / Owner / 最后复查 / 下次复查 / 可使用位置`
