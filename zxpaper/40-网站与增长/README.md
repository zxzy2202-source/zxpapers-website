# 40 网站与增长

这里负责把已验证的产品和市场知识转成 zxpapers.com 的页面与内容资产。

## 标准结构

| 模块 | 内容 |
|---|---|
| `41-网站信息架构与URL` | 页面 Owner、canonical URL、生命周期、合并/保留规则 |
| `42-页面Brief` | H1、受众、购买任务、模块、FAQ、CTA、询盘字段 |
| `43-SEO关键词与内链` | 主词、意图、页面归属、内链与冲突检查 |
| `44-GEO答案与实体` | 可引用定义、对比、FAQ、实体和证据链接 |
| `45-内容计划` | Blog、指南、案例、行业洞察和更新节奏 |
| `46-发布与复盘` | 发布检查、索引、询盘质量、30/60/90 天复盘 |

## 与当前网站对齐

首批知识包必须覆盖当前站点的四组页面：

1. Thermal paper rolls：品类、尺寸、环保体系、定制印刷。
2. Thermal labels：尺寸、shipping、barcode、linerless、定制印刷。
3. Printed/product labels：can、detergent、product labels 等聚合与细分页。
4. NCR forms：联数、表单类型、行业应用和定制页。

每个页面只允许一个关键词与知识 Owner。尚未取得证据的能力不能因为页面需要而自动升级为卖点。
